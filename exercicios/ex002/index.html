Quebra de Linhas

Para quebrar somente 1 linha no parágrafo, utilizamos a tag <br>, porem não utilizamos essa tag para quebrar mais de 1 linha.

Na quebra de multiplas linhas devemos usar o CSS

HTML Entities

HTML Entities são simbolos especiais, para adicionar não podemos simplesmente colocalos no código, precisamos utilizar da seguinte forma:
&larr = ←
&uarr = ↑
&rarr = ↑
&darr = ↓
É necessário ter em mente que todos os códigos necessítam de ";" no final.
