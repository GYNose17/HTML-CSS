Estilo

font-style

A tag font-style: ; define o estilo da fonte com os seguintes parâmetros:
normal - italic - oblique


font-family

A tag font-family: ; define a fonte, segue o exemplo: 

font-family: Arial, Helvetica, sans-serif;

Dessa maneira, a font Arial é tratada como principal, e a Helvetica só sera utilizada se o dispositivo do usuário não possuir suporte para a fonte primária, da mesma maneira se não houver suporte para a fonte secundaria ele procurará por uma fonte "sans-serif" (Sem Serifa) genérica que o aparelho tenha disponível.


font-weight

A tag font-weight: ; define a espessura da fonte e possui os seguintes parâmetros:
lighter - normal - bold - bolder
100 - 200 - 300 - 400 - 500 - 600 - 700 - 800 - 900
Os parâmetros escritos estão presentes na maioria das fontes, enquanto os parâmetros numéricos aparecem com menas frequência.

Shorthand Font

Introdução

É muito comum a utilização das tags: font-style font-weight font-size font-family. Sendo assim foi criado uma maneira de resumir tudo isso em somente uma linha de código.

Funcionamento

Utilizando a tag font: ; é necessário adicionar os parâmetros desejados de "style" "weight" "size" "family", nessa ordem, para o resumir algumas linhas de código.

Exemplo

O que seria: 

    <style>
        *{
            font-style: italic;
            font-weight: bold;
            font-size: 3em;
            font-family: 'Courier New', Courier, monospace;
        }
    </style>

Se torna: 

    <style>
        *{
            font: italic bolder 3em 'Arial', sans-serif;
        }
    </style>

Fontes Externas

Importadas

Temos apenas algumas fontes disponíveis diretamente da máquina, sendo assim para tornar possível a utilização de outras fontes é necessário importa-las.
Com o fonts.google.com, é possível adiquirir várias fontes e "linka-las" ao nosso código para utilização.

Exemplo

    <style>
        @import url('');
    </style>

Basta adicionar o link da fonte e ja esta pronta para uso.

Download

Em muitos projetos, fontes específicas são requisitadas, sendo assim é necessario baixa-las e liga-las ao servidor.
Com a fonte desejada no devido diretório, é necessário utilizar o código a seguir: 

    <style>
        @font-face {
            font-family: 'Nome da Fonte';
            src: url('[..]/Fonte Exemplo.otf') format('opentype'), url([..]/Fonte Exemplo.ttf) format('truetype');
            font-weight: normal;
            font-style: normal;
            font-size: 1em;
        }
    </style>

Dentro do url('') é colocado o diretório do arquivo e no format('') o devido código, de acordo com a tabela:

  • opentype (otf)
  • truetype (ttf)
  • embedded-opentype
  • truetype-aat (Apple Advanced Typography)
  • svg

Obs: Existem excessões

Seletores Personalizados

Para configurar itens específicos utiliza-se seletores personalizados, sendo eles, id ou class.

Id

Introdução

O grande diferencial do Id é que só se pode usar um seletor em todo o código HTML, ou seja, não pode haver dois ou mais Ids com o mesmo nome.

Exemplo

    <head>
        <style>
            h1#exemplo{
                /*
                    toda a formatação adicionada aqui modificará especialmente o h1 de id exemplo
                */
            }
        </style>
    </head>
    <html>
        <body>
            <h1 id="exemplo">exemplo</h1>
        </body>
    </html>

Class

Introdução

Pode ser utilizado várias vezes durante o código e cumpre muito bem a tarefa de estilizar partes semânticas do HTML, além disso, é possivel utilizar a tag class dentro de uma tag span, assim estilizando uma parte específicas do código.

Exemplo

<html>
<head>
    <style>
        .destaque {
            background-color: #dbdb7794;
        }
        .titulos {
            font-weight: bold;
        }
    </style>
</head>

<body>
        <h1 class="destaque titulos">Título</h1>
        <p>
            Lorem ipsum dolor sit amet consectetur adipisicing elit. Nobis, libero qui. Modi dolorum magni ad sint itaque quis ipsam. Maxime ipsum debitis enim repellendus iusto repudiandae tempora possimus dolorum ullam.
        </p>
        <h2 class="destaque">Subtítulo</h2>
        <p>
            Lorem, ipsum dolor sit amet consectetur adipisicing elit. Aliquid labore aut alias nam tempora dolorum sunt rem quos, repellat natus animi, recusandae a quod error dolore nulla voluptatum ad impedit?
        </p>
    </body>
</html>
